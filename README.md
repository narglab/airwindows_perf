The airwindows project by Chris Johnson is a vast collection of quality FOSS audio plugins.

This is airwindows_perf, a fork in construction (see my public todo-list).

# Build

You can build the plugins under airwindows_perf/JUCE/ using the Projucer.
Here is the official introduction: 
https://juce.com/tutorials/tutorial_new_projucer_project/

1. Open the .jucer file in the Projucer.
2. Add your Exporter.
3. Open the solution in your IDE and build.

You can find the plugin file in
<Plugin_Folder_Name>/Builds/<Exporter_Name>
